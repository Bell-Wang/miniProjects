Visualization on shinnyapps:
https://bella-y-wang.shinyapps.io/ny_events/


1. To set up project dependency, run:
pip install -r requirements.txt


2. Run:
Python text_mine.py

The output can be found in 'output/' directory. 'instagram_data.csv' and 'tweet_data.csv' contain full dataset with more interesting features. 'instagram_hist.csv' and 'tweet_hist.csv' are the data table for the histogram charts.
